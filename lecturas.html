GIT y GITHUB

Son sistemas de control de versiones como GIT y GITHUB, se logran compartir y publicar archivos en formato HTML5 y CSS3 a través de una línea de comandos o por medio de la consola gráfica de GITHUB desktop.

HTML5

Los sitios web en HTML5, pueden ser visualizados en diferentes navegadores teniendo en cuenta su funcionalidad por medio del uso de diferentes editores de código.

CSS3

Los sitios web en formato CSS3 pueden ser visualizados en diferentes navegadores teniendo en cuenta su funcionalidad y usabilidad por medio del uso de diferentes editores de código.
